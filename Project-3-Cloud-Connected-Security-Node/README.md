# 🔐 Project 3 – Cloud Connected Security Node

## Objective
To detect nearby objects using an HC-SR04 ultrasonic sensor and send real-time distance data to Adafruit IO using ESP32 over Wi-Fi for remote monitoring.

## Components Used
- ESP32
- HC-SR04 Ultrasonic Sensor
- Adafruit IO
- Wi-Fi
- Wokwi Simulator

## Features
- Real-Time Distance Measurement
- Wi-Fi Connectivity
- MQTT Communication
- Cloud Data Upload
- Adafruit IO Dashboard Integration
- Intruder Detection
- Safe / Warning / Intruder Status
- Serial Monitor Output

## Software Used
- Arduino IDE
- Wokwi Simulator
- Adafruit IO
