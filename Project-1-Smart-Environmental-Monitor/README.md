# 🌡️ Project 1 – Smart Environmental Monitor

## Objective
To monitor temperature and humidity using an ESP32 and DHT22 sensor.

## Components Used
- ESP32
- DHT22 Sensor
- Wokwi Simulator

## Features
- Real-time Temperature Monitoring
- Real-time Humidity Monitoring
- Serial Monitor Output

## Software Used
- Arduino IDE
- Wokwi Simulator
