# 🌱 Project 2 – Automated Irrigation Controller

## Objective
To monitor soil moisture using an ESP32 and automatically control a water pump based on soil moisture levels.

## Components Used
- ESP32
- Soil Moisture Sensor(Potentiometer)
- Relay Module (Simulated)
- Wokwi Simulator

## Features
- Real-Time Soil Moisture Monitoring
- Automatic Pump ON/OFF Control
- Dry / Normal / Wet Soil Detection
- Hysteresis-Based Control
- Serial Monitor Output

## Software Used
- Arduino IDE
- Wokwi Simulator
